# MPG (Most Popular Games)

## 달성 목표

- 영어공부
- 스프링부트
- 리엑트
- 자바스크립트
- OpenApi (Twitch)


> 내 스트리머가 많이하는 게임, 내 스트리머가 하고있는 게임
> - 게임 순위
> - 가격정보
> - 플랫폼
> - 게임이름
> - 플레이타임
> - 스트리머 명


## 트위치 API 

1. 트위치 내 한국 게임 방송 스트리머 조회
2. 선택한 스트리머 평균 시청자 수 조회
3. 국내 평균 시청자가 1000명 이상인 스트리머 조회
4. 국내 평균 시청자가 많은 스트리머 조회 (시청자 수 역순으로 200명까지)
5. 스트리머가 하고 있는 게임 조회

현재 스트리머가 하고 있는 게임 시작, 종료 시간 기록
